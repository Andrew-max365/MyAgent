## 一、基础排版参数速查

### 1.1 中文字号与磅值(pt)严格对照表
| 字号 | 磅值(pt) | 常见用途 |
| :--- | :--- | :--- |
| 初号 | 42.0 | 特大标题 |
| 小初 | 36.0 | 大标题 |
| 一号 | 26.0 | 标题 |
| 小一 | 24.0 | 标题 |
| 二号 | 22.0 | 章标题/公文标题 |
| 小二 | 18.0 | 节标题 |
| 三号 | 16.0 | 章标题/公文一级标题/公文正文 |
| 小三 | 15.0 | 节标题/公文二级标题 |
| 四号 | 14.0 | 节标题/公文页码 |
| **小四** | **12.0** | **学位论文正文/标准文档正文** |
| 五号 | 10.5 | 期刊论文正文/图表文字/页眉页脚 |
| 小五 | 9.0 | 注释/参考文献 |

### 1.2 常用中文字体规范
- **宋体 (SimSun)**：中文公文、学位论文、学术论文的**正文**首选字体。
- **黑体 (SimHei)**：中文文档的**一级、二级标题**常用字体。
- **仿宋_GB2312**：**党政机关公文**的**正文**专用字体（注意不是普通仿宋）。
- **楷体_GB2312**：公文二级标题或签发人常用。
- **Times New Roman**：英文、数字、公式中的变量首选字体。

## 二、中国学位论文通用格式（以北大标准为基准）

> 适用于硕士、博士学位论文及本科毕业设计。

### 2.1 页面设置
- **页边距**：上 3.0cm，下 2.5cm，左 2.6cm，右 2.6cm（装订线 0cm）
- **页眉页脚**：页眉距边界 2.0cm，页脚距边界 1.75cm

### 2.2 字体与段落
- **正文**：
  - 中文：宋体，**小四号 (12.0pt)**
  - 英文/数字：Times New Roman，**12.0pt**
  - 行距：**固定值 20 磅** (或 1.25-1.5 倍行距)
  - 缩进：首行缩进 2 字符
- **一级标题（章标题，如“第一章”）**：
  - 字体：黑体，**三号 (16.0pt)**
  - 对齐：**居中**
  - 间距：段前 24 磅，段后 18 磅，单倍行距
- **二级标题（节标题，如“1.1”）**：
  - 字体：黑体，**四号 (14.0pt)**
  - 对齐：左对齐
  - 间距：段前 24 磅，段后 6 磅，行距固定值 20 磅
- **三级标题（条标题，如“1.1.1”）**：
  - 字体：黑体，**13pt** (或小四号 12.0pt 加粗)
  - 对齐：左对齐
  - 间距：段前 12 磅，段后 6 磅，行距固定值 20 磅

## 三、学术期刊论文格式（GB/T 7713.2-2022 标准）

> 适用于投稿期刊的学术文章，版面较紧凑。

- **正文**：宋体，**五号 (10.5pt)**；行距：单倍或 1.2 倍。
- **一级标题**：黑体，**四号 (14.0pt)**，居中或左对齐。
- **二级标题**：黑体，**五号 (10.5pt)**，左对齐。
- **图表标题**：黑体，小五号 (9.0pt)。
- **参考文献**：宋体，小五号 (9.0pt)。

---

## 四、党政机关公文格式（GB/T 9704-2012）

> 适用于红头文件、正式报告、请示等。

- **正文**：**仿宋_GB2312**，**三号 (16.0pt)**。
- **行距**：**固定值 28 磅**（亦有 28-30 磅之说）。
- **一级标题**：**黑体**，**三号 (16.0pt)**。
- **二级标题**：**楷体_GB2312**，**三号 (16.0pt)**。
- **三级标题**：**仿宋_GB2312**，**三号 (16.0pt)**，**加粗**。
- **页边距**：上 3.7cm，下 3.5cm，左 2.8cm，右 2.6cm。
- **页码**：宋体，四号 (14.0pt)，外侧对齐。

## 五、英文学术论文格式（APA 7th / MLA 9th）

> 适用于国际期刊投稿或留学生作业。

### 5.1 APA 第7版 (最通用)
- **正文**：Times New Roman **12.0pt** (或 Calibri 11pt, Arial 11pt)。
- **行距**：**双倍行距 (2.0)**。
- **页边距**：上下左右均为 **1 英寸 (2.54cm)**。
- **缩进**：首行缩进 0.5 英寸 (1.27cm)。
- **一级标题**：**Bold (加粗)**，**Centered (居中)**，Title Case (单词首字母大写)。
- **二级标题**：**Bold (加粗)**，**Left-aligned (左对齐)**，Title Case。

### 5.2 MLA 第9版
- **正文**：Times New Roman **12.0pt**。
- **行距**：**双倍行距 (2.0)**。
- **页边距**：1 英寸 (2.54cm)。
- **无标题加粗**：MLA通常不要求标题加粗，但现代使用中常加粗以区分。

## 六、常见指令转换为 JSON 的映射表（系统核心）

> ⚠️ **注意**：此处为大模型输出 JSON 的参照标准，必须严格遵守嵌套结构！
> **单位换算提示**：小四=12.0，三号=16.0，四号=14.0，五号=10.5。

| 用户指令 | 思考逻辑（查阅上方标准） | 最终输出的 JSON 格式 (Strict Nested) |
| :--- | :--- | :--- |
| **"按学位论文/毕业论文格式排版"** | 查阅二：正文宋体小四(12pt)+固定行距20磅；一级标题黑体三号(16pt)居中 | `{"body": {"font_name": "宋体", "font_size_pt": 12.0, "line_spacing": 20, "line_spacing_rule": "exact", "first_line_indent": 2}, "heading": {"h1": {"font_name": "黑体", "font_size_pt": 16.0, "align": "center", "bold": true}, "h2": {"font_name": "黑体", "font_size_pt": 14.0, "align": "left", "bold": true}}}` |
| **"按公文格式/红头文件格式"** | 查阅四：正文仿宋三号(16pt)+固定行距28磅；一级标题黑体三号 | `{"body": {"font_name": "仿宋_GB2312", "font_size_pt": 16.0, "line_spacing": 28, "line_spacing_rule": "exact"}, "heading": {"h1": {"font_name": "黑体", "font_size_pt": 16.0, "align": "center"}, "h2": {"font_name": "楷体_GB2312", "font_size_pt": 16.0}}}` |
| **"按英文APA格式/英文论文格式"** | 查阅五：正文TNR 12pt+双倍行距；标题加粗居中 | `{"body": {"font_name": "Times New Roman", "font_size_pt": 12.0, "line_spacing": 2.0}, "heading": {"h1": {"font_name": "Times New Roman", "font_size_pt": 12.0, "bold": true, "align": "center"}, "h2": {"font_name": "Times New Roman", "font_size_pt": 12.0, "bold": true, "align": "left"}}}` |
| **"正文宋体小四，标题黑体三号"** | 直接映射：小四=12.0，三号=16.0 | `{"body": {"font_name": "宋体", "font_size_pt": 12.0}, "heading": {"h1": {"font_name": "黑体", "font_size_pt": 16.0}}}` |
| **"把标题改成红色，正文1.5倍行距"** | 简单修改 | `{"body": {"line_spacing": 1.5}, "heading": {"h1": {"color": "FF0000"}, "h2": {"color": "FF0000"}}}` |
